# Codex Handoff

Use this prompt when starting a fresh Codex session:

```text
You are building Flyto2, an open-source personal health digital twin
prototype in Go. Read AGENTS.md and PROJECT_CONTEXT.md first.

Build an MVP that imports or simulates wearable/lifestyle daily aggregates,
predicts next-day HRV, resting heart rate, fatigue, and sleep quality, compares
predictions with actual values, and explains likely missing variables.

Do not make medical, treatment, anti-aging, stem-cell, telomere, or clinical
accuracy claims. Keep all public data privacy-safe and use synthetic demo data
by default.

The public dashboard is in `web/` and uses React Vite. It reads
`web/public/public-data.json`, generated by `make web-data`.
```

## Current Follow-Up Tasks

1. Preserve the documented CLI and public JSON contracts.
2. Add JSON import only with the same finite-value, chronology, and privacy
   validation as CSV.
3. Add the weighted trend model behind `Model` and benchmark it before exposure.
4. Keep future device adapters behind equipment and privacy gates.
5. Configure and verify a real production dashboard URL before enabling search
   indexing.
6. Regenerate source references and run `make verify` after behavior changes.
